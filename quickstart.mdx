## Get Started with Firecrawl

In this quickstart, you'll learn how to:
1. Install Firecrawl
2. Scrape your first webpage 
3. Extract content in Markdown format

### Installation

Install Firecrawl using your preferred package manager:

<CodeGroup>
```bash npm
npm install @mendable/firecrawl-js
```

```bash pip 
pip install firecrawl-py
```
</CodeGroup>

### Authentication

1. Get your API key from [firecrawl.dev](https://firecrawl.dev)
2. Initialize Firecrawl with your key:

<CodeGroup>
```js JavaScript
import FirecrawlApp from '@mendable/firecrawl-js';

const app = new FirecrawlApp({apiKey: "YOUR_API_KEY"});
```

```python Python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")
```
</CodeGroup>

### Your First Scrape

Let's scrape a webpage and get its content in Markdown format:

<CodeGroup>
```js JavaScript
const result = await app.scrapeUrl('https://example.com', {
  formats: ['markdown', 'html']
});

console.log(result.markdown);
```

```python Python
result = app.scrape_url('https://example.com', {
  'formats': ['markdown', 'html']
})

print(result['markdown'])
```
</CodeGroup>

That's it! The response will contain clean markdown content ready for your AI applications:

```json
{
  "success": true,
  "data": {
    "markdown": "# Example Domain\n\nThis domain is for use in illustrative examples...",
    "html": "<h1>Example Domain</h1><p>This domain is for use in illustrative examples...</p>",
    "metadata": {
      "title": "Example Domain",
      "description": "Example Domain",
      "language": "en",
      "sourceURL": "https://example.com"
    }
  }
}
```
